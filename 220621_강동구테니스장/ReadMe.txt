이 파일은 해당 매크로를 사용하기 위한 메뉴얼입니다. 사용 전 내용을 빠짐없이 정독하셔서 실제 상황에서 원하시는 결과 있기를 바랍니다.



사용 방법)

    1. 매크로를 작동시키고 입력값을 입력한다.
    2. "일체의 조작을 하지 않고" 서버 시간이 10시가 될 때까지 대기한다.
    3. 매크로가 정상적으로 작동하는 것을 확인한다.



좌표 이용법)

    1) 날짜의 경우

        달력을 7 X 5의 xy 좌표계로 생각하시면 됩니다. "요일"이 x축, "주"가 y축 입니다.
        예를 들어 2022년 6월 1일은 (4,1), 2022년 6월 2일은 (5,1), 2022년 6월 8일은 (4,2)입니다.
        좌표 입력시에 공백이 있어서는 안됩니다. " 1,1", "1,1 ", "1 ,1", "1, 1"등은 모두 잘못된 입력입니다.
        첨부해드린 이미지를 참고해주세요.

    2) 시설과 코트의 경우

        시설은 명일테니스장과 강일테니스장으로 나뉩니다. 
        명일테니스장은 1, 강일테니스장은 2로 나타냅니다.
        코트는 코트의 번호로 나타냅니다.
        예를 들어, 명일테니스장의 3번 코트는 (1,3)
        강일테니스장의 10번 코트는 (2,10)입니다.



시간 입력 방법)

    시간의 경우 코트마다 다르게 설정되어 있어 위에서부터 순차적으로 번호를 매겨 입력을 합니다.
    이 부분은 채팅을 통해 인지하고 계신다는 답장을 받았으니 자세한 설명은 생략하도록 하겠습니다.



아이디와 비밀번호 설정 방법)

    ID&PW.txt라는 파일에 차례로 기재하시면 됩니다.
    첫 번째 행에 아이디를, 두 번째 행에 비밀번호를 입력해주시면 매크로가 스스로 인식합니다.



사용상의 주의사항)

    1. 매크로 파일을 실행하면 나타나는 검은창(이하 콘솔)에 입력할 값들을 모두 입력한 이후에는 팝업창을 끄신다거나, 다른 페이지로 이동하는 등 일체의 동작을 하시면 안됩니다.
    2. 압축을 통해 보내드린 파일은 모두 항상 같은 폴더 내에 있어야 합니다. 특히 chromedriver.exe와 marco.exe가 서로 다른 폴더에 있을 경우 작동 자체가 불가능합니다.
    3. 첨부해드린 파일 중 chromedriver.exe라는 파일이 있는데, 이것은 크롬 드라이버입니다. 
    크롬 드라이버의 버전은 항상 크롬의 버전과 일치해야합니다. 크롬이 업데이트 될 경우 크롬의 버전에 맞게 크롬 드라이버도 함께 업데이트 부탁드립니다.
    4. 매크로를 너무 일찍 실행하시면 서버 시간을 가져오는 작업 때문에 서버가 부적절한 접근으로 판단하여 차단될 수 있습니다. 매크로가 작동하지 않게 되는 주요 요인이니 58분, 59분 쯤에 실행하신 뒤 대기해주세요.
    5. 인터넷 환경은 매크로의 작동에 큰 영향을 미칩니다. 가능한 한 최고의 인터넷 환경에서 예약을 시도해주시기 바랍니다.
